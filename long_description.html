This application will help Rachael Sparks, keeper of UCL Institute of Archaeology's Collections, to get more information on the historical photographs of the Horsfield Archive. Enhancing this important photographic archive will facilitate searching specific themes such as people, certain types of architecture, archaeological sites, etc.

This application loads a picture from the Horsfield Archive and asks people to tag identified people or elements. The user can click on the picture and drag to draw a square on top of the element. This action will open a window where the user will be able to type the name of that element and save it.
